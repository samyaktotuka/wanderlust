# WANDERLUST
Wanderlust is a responsive, full-stack web application that allows users to list and browse vacation rentals. With a dynamic frontend and a robust backend, Wanderlust offers seamless property searches, secure bookings, and user-generated reviews â€” all in real time.

##  Features
-  User authentication and role-based access
-  Users can post and read reviews
-  Real-time map integration

-  Different permissions for guests, hosts, and admins
